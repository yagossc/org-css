Welcome to Org-css.

Org-css is a css theme targeted at org mode. Org-css was inspired by Imagine.

Org-css is available on Github here. To use org-css add the following headers to your org document: 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="style.css" />
#+OPTIONS: toc:nil num:nil html-style:nil

Learn more

To learn more about Org-css and what I do feel free to check out my blog or my github.
